# 指令速查（基于 riscv_assembly.v 的 Verilog “汇编器”宏）

本文件面向本教程的 Verilog 汇编器 [riscv_assembly.v](riscv_assembly.v) 中提供的寄存器别名与指令宏（task/function）。它不是 GNU 汇编语法，而是“在 Verilog 里调用 task 生成机器码”的写法。

## 寄存器类别与 ABI 约定速查

### 类别（Category）

- **特殊寄存器（Special）**
  - `zero`（x0）：Zero register（零寄存器），读恒为 0，写入被忽略
  - `ra`（x1）：Return address（返回地址寄存器）
  - `sp`（x2）：Stack pointer（栈指针）
  - `gp`（x3）：Global pointer（全局指针，常被用作 IO 基址或小数据区基址）
  - `tp`（x4）：Thread pointer（线程指针）
- **临时寄存器（Temporary, caller-saved）**
  - `t0..t6`（x5..x7, x28..x31）：Temporary register（临时寄存器）
  - 约定：调用者负责保存（函数调用可能破坏它们）
- **保存寄存器（Saved, callee-saved）**
  - `s0..s11`（x8..x9, x18..x27）：Saved register（被调用者保存寄存器）
  - `fp`（x8）：Frame pointer（帧指针），与 `s0` 同一个寄存器别名
  - 约定：被调用函数若使用这些寄存器，应在栈上保存并在返回前恢复
- **参数/返回值寄存器（Argument/Return, caller-saved）**
  - `a0..a7`（x10..x17）：Argument register（参数寄存器）；其中 `a0` 同时也是 Return value（返回值）

## 指令宏速查（每条三行）

说明：
- 下面的“用法”是本仓库 Verilog 汇编器的宏调用形式（例如 `ADD(x1,x0,x0);`），不是 GNU 汇编文本。
- `imm/offset/csr` 传入时是 32 位数，但会按 RV32I 对应格式截取/编码；`offset` 通常来自 `LabelRef(label)`。

### ADD
- 用法：`ADD(rd, rs1, rs2);`
- 英文：ADD — Add (register-register)；中文：加法（寄存器-寄存器）
- 作用：`rd = rs1 + rs2`

### SUB
- 用法：`SUB(rd, rs1, rs2);`
- 英文：SUB — Subtract；中文：减法
- 作用：`rd = rs1 - rs2`

### SLL
- 用法：`SLL(rd, rs1, rs2);`
- 英文：SLL — Shift Left Logical；中文：逻辑左移
- 作用：`rd = rs1 << (rs2[4:0])`

### SLT
- 用法：`SLT(rd, rs1, rs2);`
- 英文：SLT — Set Less Than (signed)；中文：有符号小于置位
- 作用：`rd = (signed(rs1) < signed(rs2)) ? 1 : 0`

### SLTU
- 用法：`SLTU(rd, rs1, rs2);`
- 英文：SLTU — Set Less Than Unsigned；中文：无符号小于置位
- 作用：`rd = (unsigned(rs1) < unsigned(rs2)) ? 1 : 0`

### XOR
- 用法：`XOR(rd, rs1, rs2);`
- 英文：XOR — Exclusive OR；中文：异或
- 作用：`rd = rs1 ^ rs2`

### SRL
- 用法：`SRL(rd, rs1, rs2);`
- 英文：SRL — Shift Right Logical；中文：逻辑右移
- 作用：`rd = rs1 >> (rs2[4:0])`（高位补 0）

### SRA
- 用法：`SRA(rd, rs1, rs2);`
- 英文：SRA — Shift Right Arithmetic；中文：算术右移
- 作用：`rd = signed(rs1) >>> (rs2[4:0])`（高位补符号位）

### OR
- 用法：`OR(rd, rs1, rs2);`
- 英文：OR — OR；中文：或
- 作用：`rd = rs1 | rs2`

### AND
- 用法：`AND(rd, rs1, rs2);`
- 英文：AND — AND；中文：与
- 作用：`rd = rs1 & rs2`

### ADDI
- 用法：`ADDI(rd, rs1, imm);`
- 英文：ADDI — Add Immediate；中文：加立即数
- 作用：`rd = rs1 + signext(imm[11:0])`

### SLTI
- 用法：`SLTI(rd, rs1, imm);`
- 英文：SLTI — Set Less Than Immediate (signed)；中文：有符号小于立即数置位
- 作用：`rd = (signed(rs1) < signext(imm[11:0])) ? 1 : 0`

### SLTIU
- 用法：`SLTIU(rd, rs1, imm);`
- 英文：SLTIU — Set Less Than Immediate Unsigned；中文：无符号小于立即数置位
- 作用：`rd = (unsigned(rs1) < unsigned(signext(imm[11:0]))) ? 1 : 0`

### XORI
- 用法：`XORI(rd, rs1, imm);`
- 英文：XORI — XOR Immediate；中文：异或立即数
- 作用：`rd = rs1 ^ signext(imm[11:0])`

### ORI
- 用法：`ORI(rd, rs1, imm);`
- 英文：ORI — OR Immediate；中文：或立即数
- 作用：`rd = rs1 | signext(imm[11:0])`

### ANDI
- 用法：`ANDI(rd, rs1, imm);`
- 英文：ANDI — AND Immediate；中文：与立即数
- 作用：`rd = rs1 & signext(imm[11:0])`

### SLLI
- 用法：`SLLI(rd, rs1, imm);`
- 英文：SLLI — Shift Left Logical Immediate；中文：逻辑左移立即数
- 作用：`rd = rs1 << imm[4:0]`

### SRLI
- 用法：`SRLI(rd, rs1, imm);`
- 英文：SRLI — Shift Right Logical Immediate；中文：逻辑右移立即数
- 作用：`rd = rs1 >> imm[4:0]`（高位补 0）

### SRAI
- 用法：`SRAI(rd, rs1, imm);`
- 英文：SRAI — Shift Right Arithmetic Immediate；中文：算术右移立即数
- 作用：`rd = signed(rs1) >>> imm[4:0]`

### JAL
- 用法：`JAL(rd, offset);`
- 英文：JAL — Jump And Link；中文：跳转并链接
- 作用：`rd = PC + 4; PC = PC + offset`

### JALR
- 用法：`JALR(rd, rs1, imm);`
- 英文：JALR — Jump And Link Register；中文：寄存器跳转并链接
- 作用：`rd = PC + 4; PC = (rs1 + signext(imm)) & ~1`

### BEQ
- 用法：`BEQ(rs1, rs2, offset);`
- 英文：BEQ — Branch if Equal；中文：相等则分支
- 作用：若 `rs1 == rs2`，则 `PC = PC + offset`

### BNE
- 用法：`BNE(rs1, rs2, offset);`
- 英文：BNE — Branch if Not Equal；中文：不等则分支
- 作用：若 `rs1 != rs2`，则 `PC = PC + offset`

### BLT
- 用法：`BLT(rs1, rs2, offset);`
- 英文：BLT — Branch if Less Than (signed)；中文：有符号小于则分支
- 作用：若 `signed(rs1) < signed(rs2)`，则 `PC = PC + offset`

### BGE
- 用法：`BGE(rs1, rs2, offset);`
- 英文：BGE — Branch if Greater or Equal (signed)；中文：有符号大于等于则分支
- 作用：若 `signed(rs1) >= signed(rs2)`，则 `PC = PC + offset`

### BLTU
- 用法：`BLTU(rs1, rs2, offset);`
- 英文：BLTU — Branch if Less Than Unsigned；中文：无符号小于则分支
- 作用：若 `unsigned(rs1) < unsigned(rs2)`，则 `PC = PC + offset`

### BGEU
- 用法：`BGEU(rs1, rs2, offset);`
- 英文：BGEU — Branch if Greater or Equal Unsigned；中文：无符号大于等于则分支
- 作用：若 `unsigned(rs1) >= unsigned(rs2)`，则 `PC = PC + offset`

### LUI
- 用法：`LUI(rd, imm);`
- 英文：LUI — Load Upper Immediate；中文：装载高位立即数
- 作用：`rd = imm[31:12] << 12`（本宏通常传入形如 `32'h12345000` 的值）

### AUIPC
- 用法：`AUIPC(rd, imm);`
- 英文：AUIPC — Add Upper Immediate to PC；中文：PC 加高位立即数
- 作用：`rd = PC + (imm[31:12] << 12)`（本宏通常传入形如 `32'h12345000` 的值）

### LB
- 用法：`LB(rd, rs1, imm);`
- 英文：LB — Load Byte (signed)；中文：加载字节（符号扩展）
- 作用：`rd = signext(mem8[rs1 + imm])`

### LH
- 用法：`LH(rd, rs1, imm);`
- 英文：LH — Load Halfword (signed)；中文：加载半字（符号扩展）
- 作用：`rd = signext(mem16[rs1 + imm])`

### LW
- 用法：`LW(rd, rs1, imm);`
- 英文：LW — Load Word；中文：加载字
- 作用：`rd = mem32[rs1 + imm]`

### LBU
- 用法：`LBU(rd, rs1, imm);`
- 英文：LBU — Load Byte Unsigned；中文：加载字节（零扩展）
- 作用：`rd = zeroext(mem8[rs1 + imm])`

### LHU
- 用法：`LHU(rd, rs1, imm);`
- 英文：LHU — Load Halfword Unsigned；中文：加载半字（零扩展）
- 作用：`rd = zeroext(mem16[rs1 + imm])`

### SB
- 用法：`SB(rs2, rs1, imm);`
- 英文：SB — Store Byte；中文：存字节
- 作用：`mem8[rs1 + imm] = rs2[7:0]`

### SH
- 用法：`SH(rs2, rs1, imm);`
- 英文：SH — Store Halfword；中文：存半字
- 作用：`mem16[rs1 + imm] = rs2[15:0]`

### SW
- 用法：`SW(rs2, rs1, imm);`
- 英文：SW — Store Word；中文：存字
- 作用：`mem32[rs1 + imm] = rs2`

### FENCE
- 用法：`FENCE(pred, succ);`
- 英文：FENCE — Fence (memory ordering)；中文：内存序屏障
- 作用：在多核/乱序环境下约束内存可见性（本教程多数核心不实现完整语义，主要用于生成机器码）

### FENCE_I
- 用法：`FENCE_I();`
- 英文：FENCE.I — Instruction-stream fence；中文：指令流屏障
- 作用：同步指令流与数据流（例如自修改代码后确保取指看到新内容）

### ECALL
- 用法：`ECALL();`
- 英文：ECALL — Environment Call；中文：环境调用（系统调用）
- 作用：触发一个 trap 进入更高层处理（操作系统/运行时）

### EBREAK
- 用法：`EBREAK();`
- 英文：EBREAK — Environment Break；中文：断点/中止
- 作用：触发断点 trap（教程里常用作“程序结束标记”）

### CSRRW
- 用法：`CSRRW(rd, csr, rs1);`
- 英文：CSRRW — Control and Status Register Read and Write；中文：读写 CSR
- 作用：`rd = CSR[csr]; CSR[csr] = rs1`

### CSRRS
- 用法：`CSRRS(rd, csr, rs1);`
- 英文：CSRRS — CSR Read and Set bits；中文：读 CSR 并置位
- 作用：`rd = CSR[csr]; CSR[csr] = CSR[csr] | rs1`（当 `rs1=zero` 时等价于只读）

### CSRRC
- 用法：`CSRRC(rd, csr, rs1);`
- 英文：CSRRC — CSR Read and Clear bits；中文：读 CSR 并清位
- 作用：`rd = CSR[csr]; CSR[csr] = CSR[csr] & ~rs1`

### CSRRWI
- 用法：`CSRRWI(rd, csr, imm);`
- 英文：CSRRWI — CSR Read/Write Immediate；中文：CSR 读写（立即数 zimm）
- 作用：`rd = CSR[csr]; CSR[csr] = imm[4:0]`

### CSRRSI
- 用法：`CSRRSI(rd, csr, imm);`
- 英文：CSRRSI — CSR Read/Set Immediate；中文：CSR 读并置位（立即数 zimm）
- 作用：`rd = CSR[csr]; CSR[csr] = CSR[csr] | imm[4:0]`

### CSRRCI
- 用法：`CSRRCI(rd, csr, imm);`
- 英文：CSRRCI — CSR Read/Clear Immediate；中文：CSR 读并清位（立即数 zimm）
- 作用：`rd = CSR[csr]; CSR[csr] = CSR[csr] & ~imm[4:0]`

### NOP（伪指令）
- 用法：`NOP();`
- 英文：NOP — No Operation；中文：空操作
- 作用：什么也不做（本宏展开为 `ADD(x0,x0,x0)`）

### LI（伪指令）
- 用法：`LI(rd, 32'h400000);`
- 英文：LI — Load Immediate (pseudo)；中文：装载立即数（伪指令）
- 作用：把 32 位常量装入 `rd`（内部会用 `ADDI` 或 `LUI+ADDI` 组合实现）

### CALL（伪指令）
- 用法：`CALL(LabelRef(func_));`
- 英文：CALL — Call (pseudo)；中文：调用（伪指令）
- 作用：调用目标并把返回地址写入 `ra(x1)`（本宏展开为 `AUIPC(x6,offset)` + `JALR(x1,x6,offset[11:0])`）

### RET（伪指令）
- 用法：`RET();`
- 英文：RET — Return (pseudo)；中文：返回（伪指令）
- 作用：跳转回 `ra(x1)`（本宏展开为 `JALR(x0,x1,0)`）

### MV（伪指令）
- 用法：`MV(rd, rs1);`
- 英文：MV — Move (pseudo)；中文：寄存器拷贝（伪指令）
- 作用：`rd = rs1`（展开为 `ADD(rd,rs1,zero)`）

### J（伪指令）
- 用法：`J(LabelRef(L0_));`
- 英文：J — Jump (pseudo)；中文：无条件跳转（伪指令）
- 作用：无条件跳转且不写回链接寄存器（展开为 `JAL(zero,offset)`；本宏注释里标了 TODO far targets）

### JR（伪指令）
- 用法：`JR(rs1, 0);`
- 英文：JR — Jump Register (pseudo)；中文：寄存器跳转（伪指令）
- 作用：`PC = rs1 + imm` 且不写回链接寄存器（展开为 `JALR(zero,rs1,imm)`）

### BEQZ（伪指令）
- 用法：`BEQZ(rs1, LabelRef(L0_));`
- 英文：BEQZ — Branch if Equal Zero (pseudo)；中文：等于 0 则分支（伪指令）
- 作用：若 `rs1 == 0` 则分支（展开为 `BEQ(rs1,x0,offset)`）

### BNEZ（伪指令）
- 用法：`BNEZ(rs1, LabelRef(L0_));`
- 英文：BNEZ — Branch if Not Equal Zero (pseudo)；中文：不等于 0 则分支（伪指令）
- 作用：若 `rs1 != 0` 则分支（展开为 `BNE(rs1,x0,offset)`）

### BGT（伪指令）
- 用法：`BGT(rs1, rs2, LabelRef(L0_));`
- 英文：BGT — Branch if Greater Than (pseudo)；中文：大于则分支（伪指令）
- 作用：RISC-V 无 BGT，等价于交换操作数后的 `BLT`（展开为 `BLT(rs2,rs1,offset)`）

### DATAW（数据伪指令）
- 用法：`DATAW(32'hDEADBEEF);`
- 英文：DATAW — Data Word (pseudo)；中文：写入一个 32 位常量
- 作用：把一个 32 位 word 原样写入 `MEM[memPC]`，并 `memPC += 4`（用于常量表/数据区）

### DATAB（数据伪指令）
- 用法：`DATAB("A","B","C","D");` 或 `DATAB(8'h41,8'h42,8'h43,8'h44);`
- 英文：DATAB — Pack 4 Bytes into a Word (pseudo)；中文：把 4 个字节打包到一个 word
- 作用：按 little-endian 写入当前 word：`b1`->低 8 位，`b4`->高 8 位，然后 `memPC += 4`

### Label（汇编辅助）
- 用法：`integer L0_ = 4; Label(L0_);`
- 英文：Label — Label check/annotation (assembler helper)；中文：标签（校验/标注）
- 作用：在 `BENCH` 下检查 `L0_` 是否等于当前 `memPC`，并打印应填的地址（用于手工两遍汇编/校验）

### LabelRef（汇编辅助）
- 用法：`JAL(zero, LabelRef(L0_));`
- 英文：LabelRef — Label reference to PC-relative offset (helper function)；中文：标签引用转相对偏移
- 作用：返回 `L - memPC` 作为 PC 相对 `offset`（通常为负数，编码为补码；CPU 解码会符号扩展）

### endASM（汇编辅助）
- 用法：`endASM();`
- 英文：endASM — End of assembly / validation (helper)；中文：汇编结束/校验结束
- 作用：在仿真中检查标签/错误状态，必要时 `$finish()`；建议每段程序末尾调用

##（补充）内部编码宏（通常不直接使用）

下面 6 个 task 不是“指令”，而是用于把字段拼成不同格式指令字的**内部编码器**。绝大多数情况下你只调用上面那些具体指令宏即可；但为了“覆盖 riscv_assembly.v 中全部 task”，这里也列出来。

### RType（内部）
- 用法：`RType(opcode, rd, rs1, rs2, funct3, funct7);`
- 英文：RType — R-type instruction encoder；中文：R 型指令编码器
- 作用：把 `{funct7,rs2,rs1,funct3,rd,opcode}` 写入 `MEM[memPC]`，并 `memPC += 4`

### IType（内部）
- 用法：`IType(opcode, rd, rs1, imm, funct3);`
- 英文：IType — I-type instruction encoder；中文：I 型指令编码器
- 作用：把 `{imm[11:0],rs1,funct3,rd,opcode}` 写入 `MEM[memPC]`，并 `memPC += 4`

### SType（内部）
- 用法：`SType(opcode, rs2, rs1, imm, funct3);`
- 英文：SType — S-type instruction encoder；中文：S 型指令编码器
- 作用：把 `{imm[11:5],rs2,rs1,funct3,imm[4:0],opcode}` 写入 `MEM[memPC]`，并 `memPC += 4`

### BType（内部）
- 用法：`BType(opcode, rs1, rs2, imm, funct3);`
- 英文：BType — B-type branch encoder；中文：B 型分支编码器
- 作用：把 Bimm 的各字段按规范拆分拼入指令字并写入 `MEM[memPC]`，然后 `memPC += 4`

### UType（内部）
- 用法：`UType(opcode, rd, imm);`
- 英文：UType — U-type upper-immediate encoder；中文：U 型高位立即数编码器
- 作用：把 `{imm[31:12],rd,opcode}` 写入 `MEM[memPC]`，并 `memPC += 4`

### JType（内部）
- 用法：`JType(opcode, rd, imm);`
- 英文：JType — J-type jump encoder；中文：J 型跳转编码器
- 作用：把 Jimm 各字段按规范拆分拼入指令字并写入 `MEM[memPC]`，然后 `memPC += 4`
